# FireArrow

## mod说明

一款秉持 “少即是多” 理念的原版向模组，仅新增两种实用新箭，却让冒险更具趣味！

- 爆炸箭：击中目标后触发小型爆炸，高效应对集群怪物，~~不破坏地形~~
- 火球箭：发射小火球追击敌人，自带火焰伤害与视觉特效。

全程贴合原版玩法逻辑，无需额外复杂设定，用最纯粹的新增内容，为你的弓箭增添更多战术选择～
